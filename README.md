# 🌍 Guess The Country

A mobile quiz application built with React Native where users guess countries based on clues.
The app provides instant feedback and tracks the user’s score, creating a simple and interactive learning experience.

---

## 📱 Features

* 🌍 Guess countries based on clues
* ⚡ Instant answer validation
* 📊 Score tracking system
* 🔄 Dynamic questions
* 📱 Built using React Native CLI (no Expo)

---

## 🛠️ Tech Stack

* React Native
* JavaScript
* React Hooks (useState)
